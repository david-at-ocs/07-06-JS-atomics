What's the difference between alert() and console.log()?

Alert prints text to a pop up, which the user has to click an 'ok' button before continuing. Console prints text to the Javascipt console, which is good for debugging.

What's one situation where you might use an `alert()` to tell the user something?

Alerts may be good for signaling to the user when something they've entered in a form field is wrong. It's used to make sure the user gets some information by making them click 'ok' before continuing.

What's one way that `alert()` might get obnoxious?

In the instances we used on this page. They should be used for important information only. Alerts should be used sparingly, otherwise the user may get anoyed having to click 'ok' multiple times.
