console.log( "5 == '5' is " + (5 == '5') ); //true

'==' checks for equal value, regardless of type.

console.log( "5 == 'five' is " + (5 == 'five') ); //false -- correct

Tries and fails (returns false) to equate the number 5 to the string 'five'.

console.log( "5 == '6' is " + (5 == '6') ); //false -- correct

5 doesn't equal 6, even if one is a string.

console.log( "5 === '5' is " + (5 === '5') ); //false -- correct

'===' checks for both equal value and equal type. Since 5 is not the same type as '5', it returns false.

console.log( "5 == 2 + 3 is " + (5 == 2 + 3) ); //true -- correct

Evaluates 2 + 3 as 5 and then equates 5 to 5.

console.log( "5 == 5 is " + (5 == 5) ); //true -- correct

5 = 5, 5 and 5 are equal and the same type
